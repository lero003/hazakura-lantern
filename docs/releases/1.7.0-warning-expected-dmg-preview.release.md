# Hazakura Lantern 1.7.0 Warning-Expected DMG Preview

This is a separate warning-expected DMG preview. It is not the `v1.7.0`
source-only checkpoint release.

The preview tag points at the `v1.7.0` source checkpoint commit. The attached
DMG was built from the follow-up packaging lane on `main` (`caeaf1d`), with the
app bundle version metadata kept at `1.7.0`.

This DMG is a developer-preview convenience artifact for macOS Apple Silicon /
`aarch64`. It is ad-hoc signed only. It is not Developer ID signed, not
notarized, not installer-grade, and macOS Gatekeeper warnings or rejection are
expected.

Only use this preview if you are comfortable evaluating unsigned and
not-notarized macOS preview binaries.
