# Hazakura Lantern v1.7.1 Warning-Expected DMG Preview

This is a warning-expected DMG preview for personal/local use and
cross-machine handoff.

The attached DMG is a developer-preview convenience artifact for macOS Apple
Silicon / `aarch64`. It is ad-hoc signed only. It is not Developer ID signed,
not notarized, not installer-grade, and macOS Gatekeeper warnings or rejection
are expected.

Only use this preview if you are comfortable evaluating unsigned and
not-notarized macOS preview binaries.

## Verify The Download

Download both the `.dmg` and `.dmg.sha256` assets into the same directory, then
run:

```bash
shasum -c hazakura-lantern_1.7.1_aarch64-warning-expected.dmg.sha256
```

## Local Verification Before Publication

```bash
git diff --check
swift test
swift build --disable-sandbox
script/build_and_run.sh --verify
script/build-warning-dmg-preview.sh
hdiutil verify dist/dmg/hazakura-lantern_1.7.1_aarch64-warning-expected.dmg
(cd dist/dmg && shasum -c hazakura-lantern_1.7.1_aarch64-warning-expected.dmg.sha256)
```

The DMG mounted locally. The contained `Hazakura Lantern.app` matched the
source app bundle id, version, build, and CDHash. The app bundle contains the
new `AppIcon.icns`.
